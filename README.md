# portfolio_self_introduction
 
## 🔍 프로젝트 개요
이번 프로젝트는 iOS와 안드로이드를 넘어 Flutter로 웹을 도전해본 첫 웹 프로젝트입니다. <br />기존 앱을 만들었던 경험을 바탕으로 상태 관리, 생명주기, Rive, 애니메이션 등 다양한 기술들을 사용해본 프로젝트입니다.
